数据库之SQL语法

发表于 2019-07-24 更新于 2019-07-27 分类于 数据库 阅读次数：

1.mysql开始使用

1.mysql的登录命令如下

mysql -h host -r root -p
提示 mysql为登录命令，-h后面是服务器的主机地址， 如果客户端和服务器在同一台机器上，host可以是localhost或IP地址，-u后面跟登录数据库的用户名称， 在这里是root，-p后面是登录密码

2.mysql已存在的数据库：

information_scheme: 存储元数据库，数据库对象信息，如用户信息，列信息，字符，分区等
performance_sheme: 存储数据库服务性能参数
mysql: 存储数据库用户权限信息
sys: 能够快速获取谁用了多少资源，有谁连接过

3.存储引擎分类

MyISAM: 不支持事物，不支持外键，访问速度非常快
INNODB: 支持事物， 功能强大， 但是占用更多的磁盘资源，效率稍微低一点
MEMORY: 内存型，访问速度非常快，当时不安全

4.主键与外键

1.主键约束：要求键列的数据唯一，并且不允许为空。主键分为单字段主键和多字段主键

  • 单字段主键格式

  • 多字段联合主键

2.外键约束：外键约束的主要作用是保证数据引用的完整性，定义外键后，不允许删除在另一个表中具有想关联的行。
语法规则：
[CONSTRAINT <外键名>] FROEIGN KEY 字段名1 REFERENCES <主表名> 主键列1

注意：从表的外键必须关联主表的主键，并且关联字段的数据类型必须匹配。
